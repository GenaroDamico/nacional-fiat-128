## Qué es un ADR

Un ADR documenta:
- **Contexto** — por qué había que tomar una decisión
- **Opciones** — qué alternativas se evaluaron
- **Decisión** — qué se eligió y por qué
- **Consecuencias** — qué implica esa elección
